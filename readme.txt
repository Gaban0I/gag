# Utiliser GIT sur Visual Studio Code :

installez l'extension GITLens
allez dans l'onglet "Source control"
et ouvrez le menu déroulant du même nom

• pour "Pull" tout les changements :
    allez sur ... à coté du titre du menu déroulant que l'on viens d'ouvrir et appuyer sur pull

• Pour "Commit" des changements :
    les changements seront affichés dans le menu déroulant "Changes"
    afin de "stage" les changements, il vous faudra appuyer sur le + à coté du titre du menu déroulant "Changes" ou sur chaque fichier que vous voulez "stage"
    suite à ça, rentrez le nom de votre commit dans la parti message au dessus
    puis appuyer sur ✓ à coté du titre du menu déroulant "source controle" pour commit tout les fichiers que vous avez "stage"

• Pour "Push" les "Commit" sur GIT :
    lorsque vous allez "commit" des changements, une bulle va apparaitre afin de synchroniser votre projet et le GIT, si vous cliquez dessus ça va "push" les "commit" sur GIT et "pull" le tout
